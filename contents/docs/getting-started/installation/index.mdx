Setting up Apee-i is straightforward, with options to clone the repository and run `go install` or use the convenient `npm` command. Apee-i 
requires [Node.js](https://nodejs.org/) version 18 or higher for installation.

## Clone Repo and Go install
To get started, you can clone the Apee-i repository directly from GitHub. 
<Stepper>
  <StepperItem title="Step 1: Clone the Apee-i Repository">
    Begin by cloning the Apee-i repository from GitHub:

    ```bash
    git clone https://github.com/apee-i/apee-i.git
    ```
  </StepperItem>

  <StepperItem title="Step 2: Access the Project Directory">
    After cloning, navigate into the project directory to start setting up:

    ```bash
    cd apee-i
    ```
  </StepperItem>

  <StepperItem title="Step 3: Install Package">
   Install **apee-i** using simple `go install` package 

    <Note title="Remember">
		To install **apee-i**, you need to have go 1.22 or higher for optimal performance and installation
    </Note>

    ```bash
    go install
    ```
  </StepperItem>

  <StepperItem title="Step 4: Verify Installation">
    Finally, verify if apee-i has installed itself or not.

    ```bash
    apee-i --version
    ```
  </StepperItem>
</Stepper>

## Quick Installation with npm

For a faster setup, use the `npm` command to install apee-i:

```bash
npm install -g apee-i
```
